Game Cards Gallery 

Галерея карточек с видеоиграми - личный проект для применения навыков верстки и JS.

Что представляет:
- Карточки с играми 3х4
- При клике карточка плавно раскрывается
- Повторный клик закрывает активную карточку
- При клике на другую карточку открывает соответственно новую и закрывает прошлую
- При наведении карточка сдвигается вниз с тенью
- Карточки не выходят за границы контейнера

Технологии:
- HTML5
- CSS3 
- JavaScript

Как запустить - открыть index.html в браузере.
